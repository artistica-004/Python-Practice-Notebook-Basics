## ✍️ Example Problem Format

### 📌 Problem 1: Sum of Even and Odd Numbers

**Problem Statement:**
Write a program to find the sum of even and odd numbers.

**Code:**

```python
even_sum = 0
odd_sum = 0

for i in range(1, 11):
    if i % 2 == 0:
        even_sum += i
    else:
        odd_sum += i

print("Even Sum:", even_sum)
print("Odd Sum:", odd_sum)
```

**Output:**

```
Even Sum: 30
Odd Sum: 25
```

**Explanation:**
We iterate through numbers from 1 to 10 and check whether each number is even or odd using modulus operator.
